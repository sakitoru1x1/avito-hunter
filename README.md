# Avito Hunter

Программа для автоматического мониторинга объявлений на Avito. Задаёшь запрос и город - она сама проверяет новые объявления каждые N минут и шлёт их тебе в Telegram с фото и ссылкой.

Удобно, когда ищешь что-то конкретное (б/у iPhone, велосипед, видеокарту) и не хочешь каждый час обновлять страницу вручную. Программа делает это за тебя и сразу пишет в Телегу, когда появляется новое объявление.

Инструкция рассчитана на обычных людей без опыта с кодом. Если что-то непонятно - читай внимательно, не пропускай шаги.

---

## Что она умеет

- Мониторит Avito по твоему запросу, городу и диапазону цен
- Работает в фоне и сама проверяет новые объявления каждые N минут
- Случайные задержки между запросами - чтобы Avito не заблокировал
- Умеет работать только в заданные часы и дни недели (расписание)
- Присылает новые объявления в Telegram с фото и описанием
- Автоматически решает капчу Avito через RuCaptcha (платный сервис, но дёшево)
- Сохраняет историю в базу - даже если перезапустить, ничего не потеряется
- Избранное, фильтр по рейтингу продавца, игнор-слова
- Несколько поисковых профилей - можно быстро переключаться

## Что нужно установить перед запуском

Программа написана на Python, поэтому сначала поставь три вещи. Порядок важен.

### 1. Python 3.9 или новее

- Зайди на https://www.python.org/downloads/ и скачай последнюю версию для Windows.
- При установке **обязательно поставь галочку "Add Python to PATH"** на первом экране установщика. Без неё ничего работать не будет.
- После установки открой меню Пуск, найди "Командная строка" (или PowerShell) и напиши:
  ```
  python --version
  ```
  Должно показать что-то вроде `Python 3.12.x`. Если пишет "не является внутренней командой" - Python не встал в PATH, переустанови с галочкой.

### 2. Google Chrome

- Нужен обычный браузер Chrome: https://www.google.com/chrome/
- Если уже стоит - ничего не делай. Программа использует Chrome чтобы ходить по Avito.

### 3. Git

- Скачай отсюда: https://git-scm.com/download/win
- Устанавливай со всеми настройками по умолчанию - просто жми "Next" до конца.
- Проверить: в командной строке `git --version`.

## Установка программы

1. Открой любую папку, где хочешь хранить программу (например, `C:\`).
2. В этой папке щёлкни правой кнопкой в пустое место и выбери "Открыть в терминале" (или "Open Git Bash here").
3. Введи команду:
   ```
   git clone https://github.com/sakitoru1x1/avito-hunter
   ```
   Появится папка `avito-hunter`.
4. Зайди в неё:
   ```
   cd avito-hunter
   ```
5. Поставь все нужные библиотеки (это делается один раз):
   ```
   pip install -r requirements.txt
   ```
   Если пишет "pip: command not found" - попробуй `python -m pip install -r requirements.txt`.

## Запуск

Просто запусти файл `start.bat` двойным кликом. Он автоматически подтянет последнюю версию с GitHub и откроет программу.

Если `start.bat` не работает - открой терминал в папке программы и выполни:
```
python main.py
```

## Настройка Telegram-бота

Чтобы получать уведомления в Телегу, нужно создать своего бота и узнать свой Chat ID. Это делается один раз, займёт 3 минуты.

### Создаём бота через @BotFather

1. В Телеграме найди контакт **@BotFather** (официальный бот, с синей галочкой).
2. Напиши ему `/newbot`.
3. Он попросит придумать название бота (любое, например "Мой Avito-шпион").
4. Потом попросит username - должно заканчиваться на `bot`. Например, `my_avito_spy_bot`.
5. BotFather пришлёт токен - длинная строка вроде `1234567890:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`. Скопируй её и держи в секрете.

### Узнаём Chat ID

1. Найди в Телеге контакт **@userinfobot**.
2. Напиши ему что угодно (например, "привет").
3. Он пришлёт твой ID - число вроде `123456789`. Это и есть Chat ID.

**Важно:** после создания бота напиши ему хотя бы одно сообщение в личку (просто "привет"). Иначе бот не сможет тебе ничего прислать - Telegram не даёт ботам писать первыми.

### Вставляем в программу

1. В программе зайди на вкладку "Настройки", блок "Telegram уведомления".
2. В поле "Токен бота" вставь токен из BotFather.
3. В поле "Chat ID" вставь число от @userinfobot.
4. Жми кнопку "Тест" - должно прийти тестовое сообщение в Телеге от твоего бота.

## Прокси и VPN - разбираемся раз и навсегда

Это самый запутанный момент для новичков, поэтому читай внимательно. Ниже три типичных сценария - найди свой и делай по инструкции.

### Зачем вообще нужны прокси

Avito - российский сайт, лучше всего работает с российского IP. Если ты заходишь из-за рубежа или через VPN - Avito может показать капчу, ограничить выдачу или вообще заблокировать.

Telegram (api.telegram.org) - заблокирован Роскомнадзором. Из России без VPN или прокси бот не сможет отправить сообщение.

Поэтому в программе два отдельных прокси:
- **Основной прокси** (блок "Прокси") - через него программа ходит на Avito
- **Прокси для Telegram** (блок внутри "Telegram уведомления") - через него бот шлёт сообщения

### Сценарий 1: Ты в России, без VPN

Avito работает напрямую - основной прокси не нужен, оставь пустым.
Для Telegram нужен зарубежный прокси (Германия, Нидерланды, любая страна где TG не заблокирован).

Что делать:
1. Купи зарубежный прокси (см. "Где взять прокси" ниже).
2. Впиши его в блок "Прокси для Telegram" в настройках.
3. Жми "Тест" рядом с Telegram - должно прийти сообщение.
4. Основной прокси оставь пустым.

### Сценарий 2: Ты в России и используешь VPN для всего

VPN шифрует весь трафик, значит и Avito идёт через зарубежный IP. Это плохо - Avito будет чаще показывать капчу и может ограничить выдачу.

Лучший вариант:
1. Купи российский прокси для Avito - впиши в "Основной прокси".
2. VPN выключи (он не нужен, если есть прокси для TG).
3. Купи зарубежный прокси для Telegram - впиши в "Прокси для Telegram".
4. Проверь оба кнопками "Тест".

Или если не хочешь покупать два прокси:
1. Оставь VPN включённым.
2. Прокси для Avito не нужен (VPN и так даёт доступ, но будут капчи чаще).
3. Прокси для Telegram тоже не нужен (VPN пробивает блокировку).
4. Но учти: Avito будет видеть зарубежный IP и нервничать. Рекомендуется антикапча (см. ниже).

### Сценарий 3: Ты не в России

Avito может ограничивать доступ с зарубежных IP. Telegram работает без проблем.

Что делать:
1. Купи российский прокси - впиши в "Основной прокси". Avito будет думать, что ты из России.
2. Прокси для Telegram оставь пустым - он и так работает.
3. Жми "Тест прокси" и "Тест" (Telegram) - оба должны быть зелёные.

### Где взять прокси

Платные сервисы (от 200-500 руб/мес):
- mobileproxy.space
- proxy6.net
- youproxy.ru

Тебе нужно знать: тип (HTTP или SOCKS5), хост, порт, логин, пароль. Всё это сервис даст после оплаты. В программе просто заполни соответствующие поля.

## Антикапча (RuCaptcha) - автоматическое решение капчи

Avito иногда показывает капчу - "перетащи фигуру" или "поставь галочку". Обычно это происходит, если ты часто парсишь или заходишь с подозрительного IP. Программа умеет решать капчу автоматически через сервис RuCaptcha. Без этого при капче придётся решать руками в открывшемся окне браузера.

### Что такое RuCaptcha

Это сервис, где живые люди решают капчи за деньги. Ты платишь за каждое решение. Стоит копейки: одна капча = 16 копеек. Даже если капча выпадает каждый час целые сутки, это меньше 4 рублей в день.

### Как подключить

1. Зарегистрируйся на https://rucaptcha.com (или https://2captcha.com - это тот же сервис, но международная версия).
2. Пополни баланс. Минимум зависит от способа оплаты, обычно от 50 рублей.
3. В личном кабинете найди свой **API-ключ** - длинная строка букв и цифр.
4. В программе зайди в "Настройки", блок "Антикапча".
5. Выбери сервис: RuCaptcha или 2Captcha.
6. Вставь API-ключ.
7. Жми "Тест баланса" - должно показать твой баланс.

Готово. Теперь когда Avito покажет капчу, программа автоматически отправит её на RuCaptcha, получит ответ и продолжит работу. В логе увидишь что-то вроде:

```
🤖 Решаю GeeTest v4 через RuCaptcha (id: a1b2c3d4...)
✅ Решение получено, применяю...
✅ Карточки найдены, капча пройдена!
```

### Если не хочешь платить за антикапчу

Можно работать без неё. Когда Avito покажет капчу, программа откроет окно браузера и попросит тебя решить вручную. После решения нажми OK в диалоге - и программа продолжит. Минус: нужно быть у компьютера.

### Как уменьшить количество капч

- Не парси слишком часто. Интервал 5-10 минут - нормально. 1 минута - Avito быстро заподозрит.
- Используй российский прокси для Avito. Зарубежные IP вызывают больше подозрений.
- Включи расписание работы - не парси 24/7.

## Настройка поиска

1. Вкладка "Результаты поиска".
2. Выбери город или поставь галку "Вся Россия".
3. В поле "Запрос" напиши что ищешь - например: `iphone 15 pro`.
4. В поле "Игнор" (опционально) - слова через запятую, которые должны отсутствовать в объявлении. Пример: запрос `iphone 15`, игнор `чехол, стекло, наушники` - чтобы не видеть аксессуары. Регистр не важен.
5. Цена от-до, минимальный рейтинг продавца - по вкусу.
6. Чекбоксы: "Звук" - пищит при новых, "Убрать услуги" - убирает сервисные объявления, "Авито доставка" - только с доставкой.
7. Жми "Начать" - программа парсит и показывает список.
8. Жми "Авто" - включается автоматическое обновление каждые N минут. Интервал настраивается в полях "от/до".

Все новые объявления улетают в Телегу с фото, ценой и ссылкой.

## Поисковые профили

Если отслеживаешь несколько разных вещей одновременно - сохрани их как профили:

1. Настрой поиск как обычно (запрос, цена, город, игнор).
2. Жми "Сохранить как профиль" - дай имя.
3. На вкладке "Профили поиска" можно создавать, редактировать, удалять и переключаться между ними.
4. Кнопка "Сделать активным" - при следующем запуске программы загрузится этот профиль.

## Расписание работы

Если не хочешь, чтобы программа парсила ночью:

1. Вкладка "Настройки", блок "Расписание работы".
2. Поставь галку "Работать только по расписанию".
3. Выбери дни недели и промежуток времени (например, Пн-Пт 09:00-21:00).
4. Сохрани. Автообновление будет работать только в эти часы.

## Не приходят уведомления в Telegram?

Пройди по этому чеклисту сверху вниз. Обычно проблема в одном из первых пунктов.

**1. Ты написал боту хотя бы одно сообщение?**
Открой своего бота в Телеге и напиши ему "привет". Telegram не даёт ботам писать первыми - пока ты не начнёшь диалог, бот не может тебе отправить сообщение.

**2. Токен и Chat ID правильные?**
Скопируй заново из @BotFather и @userinfobot. Частая ошибка - лишний пробел в начале или конце.

**3. Ты в России без VPN и без прокси для TG?**
Telegram заблокирован в РФ. Без зарубежного прокси или VPN бот не может достучаться до api.telegram.org. Впиши зарубежный прокси в "Прокси для Telegram" и нажми "Тест".

**4. У тебя VPN включён, но прокси для TG тоже заполнен?**
Иногда VPN конфликтует с прокси. Попробуй: выключи VPN и оставь только прокси для TG. Или наоборот - убери прокси для TG и оставь только VPN. Одного из двух достаточно.

**5. Прокси рабочий?**
Жми "Тест" рядом с Telegram. Если ошибка - проверь хост, порт, логин, пароль, тип (HTTP/SOCKS5). Попробуй другой прокси.

**6. Кнопка "Тест" работает, а при парсинге не приходит?**
Проверь, стоит ли галочка "Слать в TG" на панели поиска. Также посмотри лог (файл `app.log`) - там будет видно, отправилось ли сообщение и какая ошибка.

**7. Сообщения приходят, но без фото?**
Фото скачиваются с серверов Avito. Если используешь VPN - Avito CDN может тормозить или блокировать. Попробуй российский прокси для Avito вместо VPN.

## Часто задаваемые вопросы

**Программа не запускается, пишет "No module named..."**
Ты забыл поставить библиотеки. Открой терминал в папке программы и выполни `pip install -r requirements.txt`.

**Chrome открывается и сразу закрывается**
Обнови Chrome до последней версии. Программа сама подтянет подходящий драйвер при первом запуске.

**Объявления вообще не появляются**
Проверь: правильный ли город, не слишком ли узкий диапазон цен, не слишком ли строгий рейтинг продавца. Попробуй временно сбросить все фильтры и расширить диапазон цен.

**Avito показывает капчу постоянно**
Ты парсишь слишком часто или с подозрительного IP. Увеличь интервал до 5-10 минут, используй российский прокси, подключи антикапчу (RuCaptcha). См. раздел "Антикапча" выше.

**Можно ли использовать без Telegram?**
Да. Оставь поля токена и Chat ID пустыми - программа будет показывать новые объявления только в интерфейсе.

**Безопасно ли это? Не забанят ли аккаунт?**
Программа не логинится в твой аккаунт Avito - она просто смотрит публичные страницы поиска, как обычный браузер. Но если парсить слишком агрессивно (каждую минуту, 24/7), Avito может заблокировать IP. Используй разумные интервалы и расписание.

**Где хранятся мои данные?**
Всё хранится локально на твоём компьютере:
- `avito_data.db` - база с историей объявлений и избранным
- `settings.json` - настройки (токены, прокси, параметры поиска)
- `app.log` - лог работы программы

Ничего из этого не отправляется в интернет. Эти файлы не попадают на GitHub.

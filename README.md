# Avito Hunter S. 🎯

Программа для автоматического мониторинга объявлений на Avito. Задаёшь запрос и город - программа сама обновляет список каждые N минут и шлёт новые объявления в Telegram. Удобно, когда ищешь что-то конкретное (например, б/у iPhone в своём городе) и не хочешь каждый час обновлять страницу вручную.

*Avito Hunter S. - легаси-версия на Selenium (буква «S» от Selenium). Разработка заморожена, новые фичи только в Avito Hunter P. (Playwright): https://github.com/sakitoru1x1/avito-hunter-p*

*Инструкция рассчитана на людей без опыта с кодом. Если что-то непонятно - читай внимательно, не пропускай шаги.*

---

## Что она умеет

- Мониторит Avito по твоему запросу, городу, диапазону цен.
- Работает в фоне и сама проверяет новые объявления каждые N минут.
- Случайные задержки между запросами - чтобы Avito не заблокировал.
- Умеет работать только в заданные часы и дни недели (расписание).
- Присылает новые объявления в Telegram.
- Сохраняет историю в базу - даже если перезапустить, ничего не потеряется.
- Избранное, фильтр по рейтингу продавца, игнор-слова.
- Несколько поисковых профилей - можно быстро переключаться.

## Что нужно установить перед запуском

Программа написана на Python, поэтому сначала поставь три вещи. Порядок важен.

### 1. Python 3.9 или новее

- Зайди на https://www.python.org/downloads/ и скачай последнюю версию для Windows.
- При установке *обязательно поставь галочку «Add Python to PATH»* на первом экране установщика. Без неё ничего работать не будет.
- После установки открой меню Пуск → «Командная строка» (или PowerShell) и напиши:
  ```
  python --version
  ```
  Должно показать что-то вроде `Python 3.12.x`. Если пишет «не является внутренней командой» - Python не встал в PATH, переустанови с галочкой.

### 2. Google Chrome

- Нужен обычный браузер Chrome: https://www.google.com/chrome/
- Если уже стоит - ничего не делай. Программа использует твой Chrome чтобы ходить по Avito.

### 3. Git

- Скачай отсюда: https://git-scm.com/download/win
- Устанавливай со всеми настройками по умолчанию - просто жми «Next» до конца.
- Проверить можно в командной строке: `git --version`.

## Установка программы

1. Открой любую папку, где хочешь хранить программу (например, `C:\`).
2. В этой папке щёлкни правой кнопкой в пустое место → «Открыть в терминале» (или «Open Git Bash here»).
3. Введи команду:
   ```
   git clone https://github.com/sakitoru1x1/avito-hunter-s
   ```
   Появится папка `avito-hunter-s`.
4. Зайди в неё:
   ```
   cd avito-hunter-s
   ```
5. Поставь все нужные библиотеки (это делается один раз):
   ```
   pip install -r requirements.txt
   ```
   Если пишет «pip: command not found» - запусти `python -m pip install -r requirements.txt`.

## Запуск

Просто запусти файл `start.bat` двойным кликом. Он автоматически подтянет последнюю версию с GitHub и откроет окно программы.

Если `start.bat` не работает - открой терминал в папке программы и выполни:
```
python main.py
```

## Настройка Telegram-бота

Чтобы получать уведомления в Телегу, нужно создать своего бота и узнать свой Chat ID. Это делается один раз, займёт 3 минуты.

### Создаём бота через @BotFather

1. В Телеграме найди контакт *@BotFather* (официальный бот, с синей галочкой).
2. Напиши ему `/newbot`.
3. Он попросит придумать название бота (любое, например «Мой Avito-шпион»).
4. Потом попросит username - должно заканчиваться на `bot`. Например, `my_avito_spy_bot`.
5. BotFather пришлёт токен - длинная строка вроде `1234567890:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`. *Скопируй её и держи в секрете*, с ней управляют ботом.

### Узнаём Chat ID

1. Найди в Телеге контакт *@userinfobot*.
2. Напиши ему что угодно (например, «привет»).
3. Он пришлёт твой ID - число вроде `123456789`. Это и есть Chat ID.

### Вставляем в программу

1. В программе зайди на вкладку *«Настройки»* → блок «Telegram уведомления».
2. В поле «Токен бота» вставь токен из BotFather.
3. В поле «Chat ID» - число от @userinfobot.
4. Жми кнопку *«Тест»* - должно прийти тестовое сообщение в Телеге от твоего бота.

Если тестовое сообщение не пришло и ты в России - читай следующий раздел про прокси.

## Прокси (нужен только в России)

Роскомнадзор блокирует `api.telegram.org`, поэтому для Telegram-уведомлений нужен *зарубежный прокси*. А Avito, наоборот, лучше парсить с *российского IP* - иначе он может банить или показывать меньше объявлений.

Поэтому в программе *два отдельных прокси*:
- *Основной прокси* (блок «Прокси») - для Avito. Лучше российский.
- *Прокси для Telegram* (блок внутри «Telegram уведомления») - для бота. Только зарубежный (например, Германия, Нидерланды).

### Где взять прокси

- Платные сервисы: mobileproxy.space, proxy6.net, youproxy.ru и другие. От 200-500₽/месяц.
- Тебе нужно: *тип* (HTTP или SOCKS5), *хост*, *порт*, *логин*, *пароль*. Всё это сервис даст после оплаты.

### Как вставить

- На вкладке «Настройки» заполни поля основного прокси (Avito).
- Ниже, в блоке «Прокси для Telegram», заполни поля зарубежного прокси.
- Жми «Тест прокси» и «Тест» (рядом с Telegram) - оба должны написать «✅ ...работает».
- Если не работают - проверь логин/пароль, хост, порт, тип (HTTP/SOCKS5).

Если ты *не в России* - прокси вообще не нужны, оба блока оставь пустыми.

## Настройка поиска

1. Вкладка *«Результаты поиска»*.
2. Выбери *город* или поставь галку «Вся Россия».
3. В поле *«Запрос»* напиши что ищешь - например: `oculus quest 3`.
4. В поле *«Игнор»* (опционально) - слова через запятую, которые должны отсутствовать в объявлении. Пример: запрос `oculus quest 3`, игнор `3s` - чтобы отфильтровать Quest 3S (регистр не важен).
5. Цена от-до, минимальный рейтинг продавца - по вкусу.
6. Чекбоксы: «Звук» - пищит при новых, «Убрать услуги» - убирает сервисные объявления, «Авито доставка» - только объявления с доставкой.
7. Жми *«▶ Начать»* - программа разок парсит и показывает список.
8. Жми *«🔄 Авто»* - включается автоматическое обновление каждые N минут. Интервал настраивается в полях «от/до».

Все новые объявления улетают в Телегу с фото и ссылкой.

## Поисковые профили

Если отслеживаешь несколько разных вещей одновременно - сохрани их как профили:

1. Настрой поиск как обычно (запрос, цена, город, игнор).
2. Жми *«💾 Сохранить как профиль»* - дай имя.
3. Потом на вкладке *«Профили поиска»* можно создавать, редактировать, удалять и переключаться между ними.
4. Кнопка «✔ Сделать активным» - при следующем запуске программы будет загружен этот профиль.

## Расписание работы

Если не хочется, чтобы программа парсила ночью или в выходные - включи расписание:

1. Вкладка «Настройки» → блок «Расписание работы».
2. Поставь галку «Работать только по расписанию».
3. Выбери дни недели и промежуток времени (например, Пн-Пт 09:00-21:00).
4. Сохрани. Теперь авто-обновление работает только в эти часы.

## FAQ

*Программа не запускается, пишет «No module named...»*
Ты забыл выполнить `pip install -r requirements.txt`. Открой терминал в папке программы и выполни.

*Chrome открывается и сразу закрывается / пишет про драйвер*
Обнови Chrome до последней версии. Selenium сам подтянет подходящий драйвер при первом запуске - нужно интернет-соединение.

*В Телегу ничего не приходит, а Avito парсится нормально*
Ты в России и у тебя не настроен прокси для Telegram. Читай раздел про прокси.

*Объявления вообще не появляются*
Проверь: правильный ли город, не слишком ли узкий диапазон цен, не слишком ли строгий рейтинг продавца. Попробуй временно сбросить все фильтры.

*Avito начал показывать капчу или пустой список*
Ты слишком часто парсишь. Увеличь интервал (например, «от 5 до 10 минут»), включи расписание, используй прокси. Авито не любит роботов.

*Карточки прыгают местами после авто-обновления*
Они отсортированы по времени публикации. Если у нескольких объявлений одинаковая метка времени («сегодня») - они располагаются по времени добавления в программу.

*Можно ли использовать без Телеги?*
Да. Оставь поля токена и Chat ID пустыми - программа будет просто показывать новые объявления в интерфейсе.

## Файлы и данные

- `avito_data.db` - твоя локальная база со всей историей. Не удаляй, если не хочешь потерять избранное и историю цен.
- `settings.json` - настройки (токены, прокси, расписание). Хранится только у тебя.
- `parser.log` - логи работы программы. Туда пишется всё, что происходит.

Эти файлы в `.gitignore` и не выгружаются в репозиторий.

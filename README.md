# Avito News Parser

Парсер объявлений Avito с GUI на tkinter. Отслеживает новые объявления по заданному запросу и городу, сохраняет историю в SQLite и шлёт уведомления в Telegram.

## Возможности

- Поиск по запросу, городу, диапазону цен
- Автообновление с настраиваемым интервалом
- Случайные задержки между действиями (анти-бан)
- Расписание работы (по дням недели и часам)
- Фильтр "Авито Доставка"
- Поисковые профили - сохраняй наборы фильтров и переключайся одной кнопкой
- SQLite + история изменения цен
- Telegram-уведомления (с отдельным прокси для TG)
- HTTP/SOCKS5 прокси для обхода блокировок
- Экспорт/импорт истории

## Установка

```bash
pip install -r requirements.txt
python main.py
```

Нужен установленный Chrome. Selenium сам подтянет драйвер.

## Модули

- `main.py` - точка входа
- `gui.py` - интерфейс
- `database.py` - SQLite (объявления, профили, история цен)
- `driver.py` - менеджер Selenium-драйвера
- `telegram.py` - отправка уведомлений
- `storage.py` - экспорт/импорт JSON
- `utils.py` - вспомогательные функции (транслит, парсинг дат, расписание, задержки)
- `config.py` - константы
- `logger_setup.py` - логирование

## Структура данных

- `avito_data.db` - основная БД (SQLite)
- `settings.json` - настройки (TG-токен, прокси, расписание)
- `parser.log` - логи

Эти файлы в `.gitignore` и локальны.
